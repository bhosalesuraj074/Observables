map operator

map operator that applies a given function to each value emitted by an observable and emits the result of that function.

mapTo operator

mapTo operator that applies a given function to each value emitted by an observable and emits the result of that function.

mergeMap operator

mergeMap operator that applies a given function to each value emitted by an observable and emits the result of that function.

switchMap operator

switchMap operator is higher-order mapping operator that project each value emitted by the source observable, when A new value is emitted by the source observble, switchMap automatically unsubscribe from the previouse inner observable and subscribe to the new inner observable, effectively 'switching' to the latest inner observable
