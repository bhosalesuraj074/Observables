🛡️Types of Operator

Creation operator

creaation operator in RxJS are used to create observables from various data sources or event. They provide a way to start and manage observable streams.

Filtering operator

filtering operator in RxJS are used to filter the unwanted data and create a new observables stream that only includes the values that meet the specified criteria.

Mathematical operator

Mathematical operator allow you to apply various mathematical operation to the emiitted values, producing transformed or aggregated results.

join operator

join operators typically emission from multiple observable sources. these operator allow you to work with observables that emit values over time and combine or marge togather.

Error handling operator

Error handling operators involves managing error that can occur within observable streams.

Transformation operator

Transformation operators that modify the item emitted by the an observable stream, trasnforming the data in some way.
