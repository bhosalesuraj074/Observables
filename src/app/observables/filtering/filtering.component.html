debounceTime operator

debounceTime operator emit a notification from the source observable only after a perticular time span has passed without another source emission.

take operator

take operator is filtering operator that limit the number of values emmited by an obsevable it take specified number of value from the beginning of the obsevables sequence

takeWhile operator

takeWhile Emits values emitted by the source Observable so long as each value satisfies the given predicate, and then completes as soon as this predicate is not satisfied.
