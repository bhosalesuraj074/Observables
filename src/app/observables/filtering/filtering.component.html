debounceTime operator

debounceTime operator emit a notification from the source observable only after a perticular time span has passed without another source emission.

take operator

take operator is filtering operator that limit the number of values emmited by an obsevable it take specified number of value from the beginning of the obsevables sequence

takeWhile operator

takeWhile Emits values emitted by the source Observable so long as each value satisfies the given predicate, and then completes as soon as this predicate is not satisfied.

takeLast operator

takeLast operator take the last N emmited by the observable where 'N' is specified count

first operator

first operator emit only the first value that matches the condition and then complete the obsevable.

last operator

last operator emit only the last value that matches the condition and then complete the obsevable.

elementAt operator

elementAt operator allows you take a specified element at a given given index from the observable

filter operator

filter operator allows to you include only value from the observable that satisfy the condition. it emits value only when the condition is true otherwise it ignore the values

distinct operator

distinct operator allows you to ensure that only unique values are allowed through an observable. it emits value that are unique and ingnore value that have already been emited.

skip operator

skip operator allows you to skip first specified number values emited by an observable and start emiting the remaining values.
